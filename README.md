# AI App Store Dashboard

实时分析 ChatGPT、Gemini、Claude 的竞争态势。

## 功能特性

- 📊 **周度/月度切换** - 查看每周或每月的数据趋势
- 📈 **多维度分析** - Downloads、MAU、WAU 三大指标
- 🎯 **应用选择器** - 单独或组合查看不同应用
- 📏 **智能缩放** - Y轴自动调整，可视化更清晰
- 🌍 **全球排行** - Top 20 国家数据分析
- 📱 **平台对比** - iOS vs Android 详细对比
- 📊 **市场份额** - 实时竞争格局追踪

## 数据来源

Sensor Tower (更新至 2026年3月)

## 技术栈

- 纯 HTML + JavaScript
- Chart.js 数据可视化
- 响应式设计
